# 文明放置 (Civilization Idle)

一款类似《文明》的简化文字放置版模拟经营游戏。

## 🎮 在线游玩

访问 GitHub Pages 链接即可游玩（部署后更新）

## 📋 游戏特性

### 资源系统（8种）
- 🌾 食物 - 维持人口
- 🪵 木材 - 基础建材
- 🪨 石料 - 高级建材
- ⚙️ 金属 - 工具和武器
- 🪙 黄金 - 贸易货币
- 🔬 科研点 - 解锁科技
- 🎭 文化 - 特殊效果
- ⛪ 信仰 - 宗教系统

### 职业系统（8种）
- 🧺 采集者 - 食物+木材
- 👨‍🌾 农民 - 大量食物
- 🪓 伐木工 - 木材
- ⛏️ 矿工 - 石料+金属
- 📚 学者 - 科研点
- 🔨 工匠 - 金属+黄金
- 🙏 祭司 - 信仰+文化
- 💰 商人 - 黄金

### 建筑系统（13种）
- ⛺ 帐篷 → 🛖 茅屋 → 🏠 房屋（人口上限）
- 🚜 农场、🏭 锯木厂、🏗️ 采石场、⛏️ 矿井（自动生产）
- 🌾 粮仓、📦 仓库（储存上限）
- 📖 图书馆、⛪ 神殿、🏪 市场、🎓 学院（特殊功能）

### 科技树
1. **石器时代**：工具制作、生火
2. **农业时代**：农业、建筑学
3. **铜器时代**：采矿、石工
4. **文明初期**：文字、宗教、货币
5. **古典时代**：哲学、工程学、贸易

### 离线挂机
- 关闭页面后游戏继续运行
- 最多24小时全额收益，之后50%收益
- 重新打开时显示离线收益汇总

### 随机事件
- 丰收、火灾、商人、灵感、移民、矿脉、瘟疫、宗教热情等

## 🚀 本地运行

```bash
# 方法1：直接打开
open index.html

# 方法2：本地服务器
cd civilization-idle
python -m http.server 8080
# 访问 http://localhost:8080
```

## 💾 存档

- 自动保存到浏览器 LocalStorage
- 支持导出/导入存档代码
- 可在不同设备间转移进度

## 🛠️ 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- 无需后端服务器
- 响应式设计，支持移动端
